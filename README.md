## Scripts
### cleanup_by_file_ext
Script to cleanup files provided an extension and directory path.<br/>
If no folder is specified, the current directory will be used as default.<br/><br/>
Sample Usage:<br/>
`cleanup_by_file_ext.sh -d "~/Downloads" -e "TMP"`
